# Assignment 3 - Create MySQL Table and Create a Diagramm via XAMPP

## 1. Apache and MySQL

Start Apache and MySQL via XAMPP.

## 2. Create a MySQL Database from .cvs-files in aufgabe1
### Linux/UNIX

```
sh create_mysql_db.sh | mysql -u root -p online_shop
```

### Windows

Open Powershell and run git's *bash*

```
bash create_mysql_db.sh | C:\xampp\mysql\bin\mysql.exe -u root -p online_shop
```

## 3. Open phpMyAdmin

Go to http://localhost/phpmyadmin and download the diagramm from the Designer.
